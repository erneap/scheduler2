Site Forecast Report Editing

(Site Scheduler/Leadership only)

The forecast report is a excel-based output that allows the site leadership to track contract accomplishment. Each contract for support breaks up its accomplishment into periods of time with an allotted number of hours per employee per function. The contract probably lists a site leader, an admin person, plus several other functional areas. As an example the site lead might be allocated 1860 hours for a period from 1 Oct to 30 Sep. To define a forecast report, we need a name for it, the report's period and the day of the week each weekly period ends. Each report's task list contains a series of charge number/extensions to define the positions and hours for the period.

Example: I'm going to use this example to explain the report and its components. The site consists of a site lead, 3 shift supervisors and 30 workers in 3 workcenters, each with 3 shifts.

  • Name: Demo_23_24
  • Period: 1 Jan 2023 - 31 Dec 2023
  • Week Ends: Friday
  • Task List:
    1. Lead - 2023/SiteLead - 1940hrs - Qty: 1
    2. Shift Supervisor - 2023/sftsup - 1860hrs - Qty: 3
    3. Workcenter Worker (WC1) - 2023/wc1wkr - 1860hrs - Qty: 5
    4. Workcenter Worker (WC2) - 2023/wc2wkr - 1860hrs - Qty: 12
    5. Workcenter Worker (WC3) - 2023/wc3wkr - 1860hrs - Qty: 13
